🌱 Sobre o Projeto

O site Agro Sustentável foi desenvolvido com o objetivo de conscientizar as pessoas sobre a importância da sustentabilidade no meio ambiente e no agronegócio. O projeto apresenta informações sobre práticas sustentáveis que ajudam a preservar os recursos naturais, aumentar a produtividade agrícola e promover o desenvolvimento sustentável.

O tema está alinhado aos princípios do Programa Agrinho, que busca incentivar a educação ambiental, a cidadania e a valorização do campo.

🎯 Objetivos
Promover a conscientização sobre sustentabilidade no agronegócio.
Apresentar práticas sustentáveis aplicadas no campo.
Demonstrar a importância da preservação dos recursos naturais.
Incentivar o uso responsável da água, do solo e da energia.
Desenvolver conhecimentos em programação web utilizando HTML e CSS.
💻 Tecnologias Utilizadas
HTML5
CSS3
📄 Estrutura do Site

🌿 Importância da Sustentabilidade no Agro

A sustentabilidade no agronegócio busca equilibrar a produção de alimentos com a preservação do meio ambiente. Algumas ações importantes incluem:

Conservação do solo.
Proteção das nascentes e rios.
Uso eficiente da água.
Redução do desperdício.
Utilização de energias renováveis.
Recuperação de áreas degradadas.

Essas práticas contribuem para garantir recursos naturais para as futuras gerações.

🚀 Como Executar o Projeto
Baixe os arquivos do projeto.
Salve o código HTML em um arquivo chamado index.html.
Salve o código CSS em um arquivo chamado style.css.
Mantenha ambos na mesma pasta.
Abra o arquivo index.html em qualquer navegador.
